# Portfolio

An earlier portfolio site built with Astro. This repo captures previous iterations of my personal site before the current `devfolio` direction.

## Highlights

- Personal site experiments and layout exploration
- Astro-based marketing and portfolio pages
- Useful as a design reference for older direction changes

## Quick start

```bash
bun install
bun run dev
```

Build and preview:

```bash
bun run build
bun run preview
```
